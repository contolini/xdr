## Developer

### Minify

To create the minified build file:

```
npm run minify
```

### Clean

To remove generated files:

```
npm run clean
```

### Test

To run test fixtures:

```
npm test
```

### Documentation

To generate all documentation:

```
npm run docs
```

### Readme

To build the readme file from the partial definitions (requires [mdp][]):

```
npm run readme
```
